## Project Overview

M.A.S.H. is an IoT and AI-driven system for automated oyster mushroom cultivation, featuring:
- Smart environmental monitoring and control
- AI-powered optimization algorithms
- Mobile applications for growers and consumers
- Integrated e-commerce platform
- Real-time data synchronization
